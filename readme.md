# Summary:

A simple top down 2D game engine for making top down shooters or factorio clones written in [beef](https://www.beeflang.org/) using [raylib](http://raylib.com).

Just doing this on the side of my main project "Hell of fame". Just using this to learn some basics of game engine design and implementation. We'll see where this goes.

# Libraries included

I included M0n7y5's [raylib-beef](https://github.com/M0n7y5/raylib-beef) to the project directly so you don't have to worry about linking it.
